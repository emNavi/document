## 飞行前安全检查
### 使用BB响：
    - 1）设BB响的报警电压为3.3V
        - 按一下BB响顶部按钮，报警电压就高0.1V，高于3.9V后，重置为2.7V
        - 可以继续按几下，让其变成3.3V即可
        - 注：BB响接电池平衡口，是有金属排线的一端朝上，塑料一端朝下
        - 并且，是从左到右连接，也就是平衡口的最左侧对上BB响的最左侧
    - 2）检测电池电压
        - 以防止运动中途电池因过放而断电，导致飞机落地造不必要的硬件损伤
        - 若BB响插上电池平衡口后，BB响一直响(BB响报警电压为3.3V)，或者总电压低于21V时，则需换电池

![2](https://emnavi-doc-img.oss-cn-beijing.aliyuncs.com/emnavi_assets/X280/Flight_Guide/2.JPEG)

### 检查飞机遥控器连接正常：
遥控器如下图所示：

![3](https://emnavi-doc-img.oss-cn-beijing.aliyuncs.com/emnavi_assets/X280/Flight_Guide/3.JPEG)

- 检查解锁情况
    - 1）油门摇杆，拉到最低！
    - 2）四个拨杆拨到最上方
    - 3）测试解锁功能
        - 将解锁键拨到最下方，飞机能转动桨叶
        - 再将其拨回到最上方，桨叶停转即可
    - 4）若解锁失败，桨叶不转
        - 将模式拨到中间，将解锁拨杆拨到最下，再拨到最上
        - 仍然不转，将模式拨杆拨到最下，解锁键从上拨到最下
    - 5）电机转动后
        - 将四个的拨杆都拨到最上方
- 每次起飞前检查急停拨杆
    - 1）将急停拨杆拨到最下方后，拨解锁杆到最下方，飞机桨叶不转
    - 2）但拨回最上方桨叶又转，证明遥控器能急停
    - 3）将四个的拨杆都拨到最上方，油门摇杆拉到最低
- 飞机上电时应注意
    - 1）手不要放到到桨保内侧
    - 2）飞机水平放置地面后，再上电
    - 3）遥控器四个拨杆，拨到最上方，而油门摇杆应拉到最低
