这是一个方便管理的小工具，能够帮助用户从个人电脑快速访问飞行器，并提供一些常用功能：
1. 为机载电脑提供基础的局域网自动发现功能
1. 集成ttyd，提供基于网页的ssh访问
1. 快速进行wifi设置
1. 可通过在机载电脑附近设置一个名为`emNaviHijack`的wifi, 诱导机载电脑的wifi连接至`emNaviHijack`
