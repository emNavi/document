## 常用脚本


:::warning
脚本需在安装了`emnaviboard`的设备上上运行， 运行完成请执行 `sync` 指令后，再执行关机操作
:::

```bash
# 重置root的密码为 emNavi
sudo /opt/emnaviboard/emnvboard-env/bin/python -m emNaviBase.init.reset_default_root_passwd
# 清除缓存，制作系统快照前使用
sudo /opt/emnaviboard/emnvboard-env/bin/python -m emNaviBase.init.clean_cache
# 设置hostname
sudo /opt/emnaviboard/emnvboard-env/bin/python -m emNaviBase.init.rename_hostname --prefix emNavi --drone_type X280 --custom_id 1
# 修改用户名
sudo /opt/emnaviboard/emnvboard-env/bin/python -m emNaviBase.init.rename_username old_name new_name
# 重置密码
sudo /opt/emnaviboard/emnvboard-env/bin/python -m emNaviBase.init.reset_passwd emnavi 123456
```
