## ubuntu 安装

下载符合你处理器架构的安装包(建议下载最新的安装包)。  
例如，ubuntu系统的X86架构:下载emnavi-discover_0.2.3_amd64.deb  
```bash
sudo  dpkg -i emnavi-discover_0.2.3_amd64.deb    #安装emnavidiscover  
```
安装完成后，电脑上增加名为 `emnavi-discover`的应用
